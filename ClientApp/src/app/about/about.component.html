About this application

This application provides a simple Pokedex (Pokemon catalog) features. It stores basic statistics about Pokemon and allows the user to add, edit, delete and view the list of Pokemon it houses.:

This simple application was built using the following core building blocks:

  • ASP.NET Core (v2.2.0), Entity Framework Core (v2.2.4) and C# for the middle-tier
  • Angular (v6.1.0) and TypeScript for client-side code
  • Bootstrap for UI layout and styling
  • Amazon Redshift for backend storage

This application was generated using .NET Core SDK version 2.2.203 using the Angular template (dotnet new angular).

The exposed Web API can be viewed via Swagger documentation via Swashbuckle. The default URL is http://localhost:5000.

The GitHub respository includes a Dockerfile for creating a Docker image.
